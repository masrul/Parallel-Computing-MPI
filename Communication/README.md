## Peer-to-Peer communication in MPI 


### Description:
Different collective communication models were implemented, e.g.,

+ **All-to-All**
    + Recursive doubling 
    + Ring 
    + Naive non-blocking 

&nbsp;
+ **All-to-All personalized**
    + E-Cube routing 
    + Hypercube routing 
    + Naive non-blocking
    + Naive wrap-around
